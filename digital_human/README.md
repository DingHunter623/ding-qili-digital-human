# 丁启利职业能力 AI 数字人资产包

本目录用于承载“丁启利职业能力 AI 数字人”的可复用资产，来源包括职业履历、证件照和 ChatGPT 应用与精益生产实践学习成果纪念证书。

## 文件清单

- `index.html`：本地职业能力展示页，可直接用浏览器打开。
- `ding-qili-digital-human.md`：数字人的完整人设、能力边界、回答风格和可复制系统提示词。
- `ding-qili-knowledge-base.json`：结构化履历知识库，适合导入到知识库、Agent、RAG 或网页组件。
- `ding-qili-ai-avatar.png`：职业化 AI 数字人头像。
- `ding-qili-source-portrait.png`：原始证件照备份。
- `chatgpt-lean-certificate.png`：ChatGPT 应用与精益生产实践学习成果纪念证书素材。

## 数字人定位

丁启利数字人定位为制造运营改善与工程技术管理专家，面向制造企业精益转型、VSM 价值流分析改善、工程管理升级、智能工厂规划、PQCD 改善和跨部门项目交付场景，能够基于个人经历与项目数据进行职业介绍、项目复盘、面试问答、改善方案初步沟通和案例表达。

## 使用建议

1. 用 `ding-qili-digital-human.md` 作为系统提示词或角色设定。
2. 用 `ding-qili-knowledge-base.json` 作为事实来源，避免数字人编造履历外信息。
3. 用 `index.html` 做个人展示、投递链接、面试沟通或内部演示。
4. 对外发布前，建议再次确认电话、邮箱、证书编号和时间信息是否需要隐藏或脱敏。
