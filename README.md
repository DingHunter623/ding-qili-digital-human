# QilyLean

**制造改善实践交流平台｜Manufacturing Improvement Platform**

QilyLean 是丁启利围绕制造业工程管理、精益改善与数智化工厂建设持续沉淀的专业平台，聚焦现场问题诊断、改善方案设计、Pilot 验证、跨部门推进与机制固化。

## 核心方向

- 精益生产（Lean Manufacturing）
- 工业工程（Industrial Engineering）
- VSM 价值流分析
- SMED 快速换型
- 标准工时、UPPH、OEE 与线平衡
- MES / ERP / APS 协同
- 数智化工厂与新工厂规划
- 目视化项目设计与交付
- AI for Manufacturing

## 平台内容

- 制造改善与工程管理能力主页
- 汽车电子、PCBA、半导体及小家电制造实践
- VSM、SMED、IE 标准工时与 OEE 改善经验
- 数智化工厂、MES、ERP、APS 与生产运营协同
- 项目案例、方法工具、知识分享与持续更新
